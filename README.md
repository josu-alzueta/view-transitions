# View Transitions

El repositorio está dividido en tres ejemplos:

- `index.html`:

  Muestra un ejemplo con `javascript`.

- `from.html` y `to.html`

  Muestran un ejemplo con `css` entre páginas.

- Directorio `react`:

  Replica el primer ejemplo pero dentro del ecosistema React, utilizando `custom hooks`.
